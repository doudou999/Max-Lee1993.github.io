最大子列和问题的四种算法实现

发表于 2017-09-13 | 分类于 数据结构与算法

根据不同的时间复杂度用C语言实现的4种算法，以最简单的6个数的最大子列和为例。代码均为自己实现，测试通过。其中有详细的注释，相信不难理解，下面我将这四种实现代码一一列举出来。

问题陈述：

　　给定N个整数的序列{A1, A2, … , AN}，求函数ƒ(i, j) = max{0, Ai + Ai+1 + … + Aj}(1<=i<j<=N)的最大值。

问题分析：

　　求给定数列的最大子列和。

三层循环嵌套，时间复杂度为O(n^3)

/*作者：Max
  版本：v1.0
 时间：2017-9-11 */ 

#include<stdio.h>
#include <time.h>
int Array[6] = {-2,11,-4,13,-5,-2};

/*函数功能：最大子列和时间复杂度最高的算法*/ 
int MaxSubseqSum1(int *array)
{
    int i,j,k;
    int MaxSum = 0;
    int ThisSum; 

    for(i=0;i<6;i++)
    {
        for(j=i;j<6;j++)
        {
            ThisSum = 0;//进入for循环初始化，忽略这个点，打脸 
            for(k=i;k<=j;k++)    
                ThisSum += array[k];

            if(ThisSum>MaxSum)
                MaxSum = ThisSum;
        } 
    }
    return MaxSum;
}

/*主函数*/
void main()
{
    int value;
    int i = 1000000;
    clock_t start, finish;
double  duration;

start = clock();
while(i)
    {
    value = MaxSubseqSum1(&Array);
    i--;
}
finish = clock();

duration = (double)(finish - start)/CLOCKS_PER_SEC;//将结果转化成以 
printf("算法1执行100万次运行时间： %f seconds\n", duration);
    printf("最大子列和为：%d\n",value);
}

算法1的时间复杂度最高，实现方式是使用三种循环嵌套，对所有的子序列求和，在其中找到最大的。在我的计算机上跑100万次的时间大概是0.156s

两层循环嵌套，时间复杂度为O(n^2)

        int i,j;

        int ThisSum;  
        for(i=0;i<6;i++)
    {

算法2比较好在于没必要每次都从头加起来，少一层循环。在我的计算机上跑100万次的时间大概是0.069s

分而治之，时间复杂度为O(nlogn)

/*作者：Max
  版本：v1.0
 时间：2017-9-12 */
#include<stdio.h>
#include <time.h>

//int Array[6] = {-2,11,-4,13,-5,-2};

int MaxSubseqSum3(int left, int right)
{

    //int mid = 2;
    int Array[6] = {-2,11,-4,13,-5,-2};

    if(left == right)
return Array[left];

//划分为两个规模更小的问题
int mid = left + right >> 1;//右移1位，相当于除以2 
    int Left_Ans = MaxSubseqSum3(left, mid);//递归调用，解决左侧的问题 
int Rig_Ans = MaxSubseqSum3(mid + 1, right);//递归调用，解决右侧的问题 

//横跨分割点的情况
int sum = 0; 
    int Left_Max; //右侧最大值  
    int Rig_Max;//左侧最大值
int i,j;
for(i = mid; i >= left; i--)  
    {
        Left_Max = Array[mid]; //右侧最大值  
sum += Array[i];
if(sum > Left_Max) 
        Left_Max = sum;
}
sum = 0;
for(j = mid+1; j <= right; j++) 
    {
        Rig_Max = Array[mid+1];//左侧最大值
sum += Array[j];
if(sum > Rig_Max) 
        Rig_Max = sum;
}

//答案是三种情况的最大值
int ans = Left_Max + Rig_Max;
if(Left_Ans > ans) 
        ans = Left_Ans;
if(Rig_Ans > ans) 
        ans = Rig_Ans;


    return ans;
} 

/*主函数*/
void main()
{
    int value;
    int i = 1000000;
    clock_t start, finish;
double  duration;

start = clock();
while(i)
    {
    value = MaxSubseqSum3(1,5);
    i--;
}
finish = clock();

duration = (double)(finish - start)/CLOCKS_PER_SEC;//将结果转化成以秒为单位 
printf("算法1执行100万次运行时间： %f seconds\n", duration);
    printf("最大子列和为：%d\n",value);
}

算法三的特点是使用了递归的方法，将一个问题不断切割，从而降低算法复杂度 ，在我的计算机上跑100万次的时间大概是0.069s。这里的实现方式貌似有点问题，时间复杂度并没有降低，欢迎大虾指正。

在线处理，时间复杂度为O(n)

/*作者：Max
  版本：v1.0
 时间：2017-9-12 
*/
#include<stdio.h>
#include <time.h>

int Array[6] = {-2,11,-4,13,-5,-2};

int MaxSubseqSum4(int *array)
{
    int MaxSum = 0;
    int ThisSum = 0;
    int i;
    for(i=0;i<6;i++)
    {
        ThisSum += array[i];
        if(ThisSum>MaxSum)
        MaxSum = ThisSum;
        else if(ThisSum<0)
            ThisSum = 0;

    }
    return MaxSum;
}

void main()
{
    int value;
    int i = 1000000;
    clock_t start, finish;
double  duration;

start = clock();
while(i)
    {
    value = MaxSubseqSum4(&Array);
    i--;
}
finish = clock();

duration = (double)(finish - start)/CLOCKS_PER_SEC;//将结果转化成以秒为单位 
printf("算法1执行100万次运行时间： %f seconds\n", duration);
    printf("最大子列和为：%d\n",value);
}

算法4最大特点是变计算边判断，不行就直接丢弃掉，该算法是解决此问题效率最高的算法。在我的计算机上跑100万次的时间大概是0.016s
